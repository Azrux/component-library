# Component Library

A modern component library built with Storybook, TypeScript, HTML, and CSS.

## Getting Started

### Prerequisites

- Node.js and npm

### Installation

```bash
npm install
```

### Development

```bash
npm run storybook
```

## Technologies

- **Storybook** - Component development and documentation
- **TypeScript** - Type-safe JavaScript
- **HTML** - Markup structure
- **CSS** - Styling
